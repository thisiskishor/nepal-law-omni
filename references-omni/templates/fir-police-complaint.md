## WHAT IS AN FIR (JAHERI)?

A First Information Report (जाहेरी / Jaheri) is a formal written complaint filed with the police regarding a cognizable offense. Under the **Muluki Criminal Procedure Code 2074**:

- Any person with knowledge of an offense may file an FIR
- Police **must** register the FIR — they cannot refuse without legal grounds
- If police refuse: complain to the District Superintendent of Police (DSP), or file directly before the District Court
- Filing a false FIR is itself a criminal offense — only file truthful complaints

## OFFENSES REQUIRING FIR (Cognizable — Police can act without court warrant)

| Category | Examples |
|---------|---------|
| Offenses against persons | Murder, assault, rape, kidnapping, domestic violence |
| Offenses against property | Theft, robbery, fraud, extortion, arson |
| Cyber offenses | Online fraud, hacking, cyberstalking (ETA 2063, Sec. 44–47) |
| Financial fraud | Cheque dishonor with intent, investment fraud, Ponzi schemes |
| Public order | Rioting, unlawful assembly |

## IF POLICE REFUSE TO REGISTER THE FIR

| Step | Action |
|------|--------|
| 1 | Request written reason for refusal |
| 2 | Complain to the District Superintendent of Police (DSP) |
| 3 | File complaint directly before the District Court under Muluki Criminal Procedure Code 2074 |
| 4 | Seek writ of Mandamus from High Court if systemic refusal |
