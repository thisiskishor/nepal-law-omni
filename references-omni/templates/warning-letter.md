# Employee Warning Letter Templates
# Governing Law: Labour Act 2074, Secs. 42–45; Labour Regulation 2075
# CRITICAL: Written warnings are mandatory prerequisites before termination for
#           non-misconduct grounds. Without documentation, termination = wrongful.
# Live-verify at: lawcommission.gov.np | dol.gov.np

---

## THE DISCIPLINARY PROCESS CHAIN UNDER LABOUR ACT 2074

```
Verbal Counseling (not mandatory but recommended)
        ↓
First Written Warning
        ↓
Second Written Warning (Show Cause Notice)
        ↓
Departmental Inquiry (for serious misconduct)
        ↓
Termination (only after above steps for non-gross-misconduct)
```

**Skipping steps = wrongful termination risk.** Always document each step.
