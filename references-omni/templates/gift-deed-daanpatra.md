## KEY RULES FOR GIFTS UNDER MULUKI CIVIL CODE 2074

| Rule | Provision |
|------|----------|
| Donor must be the lawful owner | Cannot gift what you don't own |
| Donor must be 18+ and of sound mind | Incapacity = void gift |
| Gift must be voluntary | Coercion or undue influence = voidable |
| Written deed mandatory for immovable property | Unwritten gift of land = unenforceable |
| Registration at Malpot mandatory for land | Unregistered land gift = invalid against third parties |
| Once registered, gift generally irrevocable | Very limited grounds for revocation |
| Reduced stamp duty for close family gifts | Verify current rates — changes with Finance Act |
| Donor cannot gift property needed for own maintenance | Muluki Civil Code 2074 protection |
