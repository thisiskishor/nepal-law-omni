# Muluki Civil Code 2074 (National Civil Code 2017 AD) — Structural Summary
# Full title: Muluki Dewani Sanhita 2074 (मुलुकी देवानी संहिता २०७४)
# Live-verify at: lawcommission.gov.np | rajpatra.dop.gov.np

---

## Overview

The Muluki Civil Code 2074 replaced the old Muluki Ain (1963 AD) and consolidated Nepal's civil law into a single unified code. It governs the legal relationships between private persons covering property, contracts, family, succession, and civil wrongs.

**Commencement:** Bhadra 17, 2075 BS (September 1, 2018 AD) — the Code came into effect alongside the Muluki Civil Procedure Code 2074.

## Part 4 — Inheritance & Succession Key Provisions

### Types of Property
| Type | Nepali | Rules |
|------|--------|-------|
| Ancestral Property | पैतृक सम्पत्ति | Partitioned among heirs; equal rights |
| Self-Acquired Property | स्वअर्जित सम्पत्ति | Owner may dispose freely; testate/intestate |

### Heirs (Priority Order — Intestate)
1. Spouse + Children (equal shares)
2. Parents
3. Siblings and their descendants
4. Other relatives (per code provisions)

**Daughters' Rights:** Equal succession rights with sons — a major reform in the 2074 Code.

**Partition (Ansa-Banda / अंशबण्डा):**
- Any co-heir may demand partition at any time
- Requires notice to all co-heirs
- Unregistered partition deed still valid between parties; registration recommended

**Wills (Wasiyatnama / वसियतनामा):**
- Must be in writing
- Signed by testator + 2 witnesses
- Can be registered at local government or Notary
- Cannot defeat compulsory share of surviving spouse and minor children

## Part 5 — Law of Contracts (Most Frequently Applied)

### Valid Contract Requirements
```
1. Offer + Acceptance (प्रस्ताव + स्वीकृति)
2. Consideration (प्रतिफल) — must be lawful
3. Capacity — both parties must be of legal age (18+) and of sound mind
4. Free Consent — no coercion, undue influence, fraud, or misrepresentation
5. Lawful Object — purpose must not violate law or public policy
```

## Part 6 — Law of Torts (Civil Wrongs)

| Tort | Standard | Compensation Basis |
|------|----------|-------------------|
| Negligence | Duty of care + breach + causation + damage | Actual loss + foreseeable damage |
| Nuisance | Unreasonable interference with use/enjoyment of property | Damages + injunction |
| Defamation | False statement + publication + damage to reputation | Pecuniary loss + non-pecuniary |
| Trespass | Unauthorized entry/interference | Nominal or actual damages |
| Strict liability | Hazardous activity — no fault required | Full compensation |
